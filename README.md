## Project Overview
This project is part of the Google Data Analytics Certificate (Course 2).
It focuses on creating, formatting, sharing, and commenting on a spreadsheet
using Microsoft Excel to record sales data for a local bakery.

## Tool Used
- Microsoft Excel

## Dataset Description
The dataset contains sample sales records for a bakery, including:
- Date of transaction
- Product name
- Price
- Quantity

The data was created for learning purposes.

## Tasks Completed
- Created a new Excel spreadsheet
- Added and formatted column headers
- Entered and formatted sales data
- Inserted, moved, and deleted rows and columns
- Added comments for documentation
- Prepared the file for sharing and collaboration
- Generated a shareable link
- Stopped sharing access for specific collaborators

## Skills Demonstrated
- Spreadsheet creation and formatting
- Data organization and consistency
- Basic data management in Excel
- Collaboration and documentation concepts
- Control over file access and commenting

## Notes
This is a guided course activity intended to demonstrate foundational spreadsheet
and collaboration skills.
